完成github托管到本地后

1.https://github.com/gohugoio/hugo/releases下载对应版本的hugo

2.将hugo.exe加入到环境变量

3.执行以下命令

hugo version // 检查安装
hugo new site my 新建站点
cd my
进入站点

4.进入themes目录下载主题，这里我下的是hyde

cd themes
https://github.com/htr3n/hyde-hyde.git
cd ../

5.本地运行

hugo server --theme=hyde

执行完后可以在本地localhost:1313运行

6.部署到github

hugo --theme=hyde --baseUrl="https://username.github.io" --buildDrafts // 生成页面模板

然后进入public文件下 将文件push到github的username.github.io仓库
